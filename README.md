# ip-info
Утилита для пробива айпи адреса ( парсит сайт ipinfo.io )

Установка в термукс
__________________________
pkg update

pkg upgrade

pkg install git


pkg install python
__________________________

pip install urllib.request

git clone https://github.com/luka-4EBUREK/ip-info/

cd ip-info
__________________________

Запуск:
python ip-info.py
